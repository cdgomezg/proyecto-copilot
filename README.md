# proyecto-copilot
Proyecto de ejemplo: sitio web informativo sobre la Metodología Ágil SCRUM.

Contenido del repositorio:

- `index.html` — página principal informativa sobre SCRUM.
- `css/styles.css` — estilos responsivos.
- `js/script.js` — pequeñas interacciones (menú móvil y scroll suave).

Cómo ver el sitio localmente:

1. Abre el archivo `index.html` en tu navegador (doble clic o "Abrir con").
2. (Opcional) Para un servidor local simple, desde PowerShell puedes ejecutar:

```powershell
# PowerShell: sirve los archivos en http://localhost:8000
python -m http.server 8000; Start-Process http://localhost:8000
```
