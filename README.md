
## Despliegue en GitHub Pages

Este repositorio está preparado para publicarse en GitHub Pages desde la rama `main` (root).

Pasos rápidos:
1. Asegúrate de que el repo remoto está en GitHub (ejemplo: `https://github.com/cdgomezg/proyecto-copilot`).
2. En la interfaz web de GitHub, ve a `Settings` → `Pages`.
3. En `Source`, selecciona `branch: main` y carpeta `/ (root)`.
4. Guarda. En unos minutos la página estará disponible en:

```
https://cdgomezg.github.io/proyecto-copilot/
```
